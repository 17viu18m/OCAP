Operation Capture And Playback
Addon v0.5.1

Installation

  1. Extract ocap.zip to your Arma 3 mods directory.
  2. Open @ocap, copy & paste 'userconfig' folder into your Arma server's root folder.
  3. Navigate to A3 root > userconfig > ocap. Open config.hpp in a text editor (e.g. notepad). Configure settings accordingly.
Repeat steps 2 to 3 for each Arma server you wish to run OCAP on.


Running OCAP

To run OCAP on your server, add @ocap to your server's '-mod' startup parameter.
e.g. -mod=@ocap;

Capture automatically begins when server becomes populated (see userconfig for settings).
To end and export capture data, call the following (server-side):
[] spawn ocap_fnc_exportData;

Tip: You can use the above function in a trigger.
e.g. Create a trigger that activates once all objectives complete. Then on activiation:
if (isServer) then { _export = [] spawn ocap_fnc_exportData; }; "end1" call BIS_fnc_endMission; // Ends mission for everyone


Note: For every 10 'frames' captured, OCAP will print a line in the Arma RPT log file. When exporting, OCAP also creates its own log file (in Arma server root folder) called ocap_log.txt - use this when debugging.
